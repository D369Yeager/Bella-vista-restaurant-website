# 🍝 Bella Vista Restaurant Website

A modern and responsive restaurant landing page for **Bella Vista**, an Italian cuisine restaurant.
The website is built using **HTML5** and **CSS3** with a clean user interface, responsive layout, smooth hover effects, animations, and a working live deployment on Netlify.

## 🌐 Live Demo

🔗 [View Live Website](https://bucolic-sunshine-091fda.netlify.app/)

## 📌 Project Overview

**Bella Vista** is a frontend restaurant website designed to present an authentic Italian restaurant experience.
It includes a beautiful hero section, fixed navigation bar, menu section, about section, contact details, reservation form, and footer.

This project was created to practice frontend development concepts such as semantic HTML, CSS styling, Flexbox, Grid, responsive design, forms, animations, and clean page structure.

## 🛠️ Tech Stack

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* CSS Animations
* Netlify Deployment

## ✨ Features

* Fixed navigation bar
* Full-screen hero section with background image
* Call-to-action buttons
* Menu section with food items and prices
* About section with restaurant image
* Contact section with address, phone number, and email
* Reservation form with name, email, phone, date, time, guests, and special requests
* Responsive layout for tablets and mobile screens
* Footer with quick links, opening hours, and social links
* Smooth hover effects and simple animations

## 📸 Website Sections

### 🏠 Home

A full-screen landing section with a restaurant background image, welcome message, and buttons to view the menu or make a reservation.

### 🍽️ Menu

A clean menu section displaying Italian dishes with descriptions and prices.

### 📖 About

An about section that introduces Bella Vista and includes a restaurant-related image.

### 📞 Contact

A contact section containing restaurant details and a reservation form.

### 🔻 Footer

Footer includes restaurant description, quick links, opening hours, and social media links.

## 📱 Responsive Design

The website uses media queries to adjust the layout for smaller screen sizes.
On mobile devices, the navigation, buttons, about section, and contact section adjust for better readability and usability.
